The Data

We used the University of Sussex-Huawei Locomotion (SHL) dataset. This is a versatile annotated dataset for multimodal locomotion analytics of mobile users. This data consists of movement measurement spanning across many days with information such as mode of trasnportation, temperature, location, etc. To use the data we converted the txt files to csv files and then uploaded those files to a MongoDB database. Utalizing NoSQL queries and PySpark functions, we have been able to collect various levels of information that can be found through this website.

Project Goals

With this project, we set out to use the University of Sussex-Huawei Locomotion (SHL) dataset to create predictions based on the user's activities. Our objective was to leverage this rich dataset, which captures various locomotion patterns, to develop advanced analytics and models that can identify trends in user behavior.
